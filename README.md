# Dream Foundation Youth Empowerment Platform

## Overview
A full-featured web platform for a youth empowerment non-profit, featuring a public-facing website and secure admin dashboard with event management, donation tracking, and user administration.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (MongoDB Atlas)
- **Libraries**: Nodemailer, bcryptjs, Multer

## Features
- Responsive public-facing website with program information and donation options
- Secure admin authentication with session management
- Event management (CRUD operations)
- Automated email notifications
- Admin dashboard with statistics
- User and donation management
